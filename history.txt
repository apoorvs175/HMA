1. Project Overview

Project Name: HostelFlow
Type: Full Dynamic Hostel Management System
Architecture: Full Stack Web Application

The HostelFlow portal is a complete hostel management platform designed to manage hostels, rooms, students, owners, wardens, finances, and system operations in a fully dynamic environment where all modules are interconnected.

The system ensures that all data changes reflect live across the entire portal.
For example:

If a hostel is created → it appears in owner and warden records

If a student is assigned to a room → room occupancy updates automatically

If an owner is updated → all related pages reflect updated data

The portal follows a live data architecture, meaning no static or hardcoded data should exist in the system.

All information must be stored in the database and dynamically fetched everywhere.
